Photoshop

Photoshop es una plataforma de fotografía, edición y diseño. Durante el curso se introdujeron las nociones básicas que permitieron entre algunas cosas la combinación de fotografías.

Illustrator

Illustrator es un editor de gráficos vectoriales; está destinado para crear y diseñar imágenes. En el curso se presentaron algunas herramientas de tal forma que como resultado se logro la vectorización de un retrato.

Dreamweaver

Dreamweaver es una aplicación informática destinada a la construcción, diseño y edición de sitios y aplicaciones Web. El curso de producción multimedial, permitió un primer acercamiento al mundo de la creación de páginas Web brindando los conocimientos bases del uso de HTML y CSS.

Spark

Adobe Spark es un conjunto integrado de aplicaciones de creación de medios para dispositivos móviles y web. En el curso se implemento principalmente para la creación de videos y gráficos sociales.

Machine Learning

Machine learning es un subcampo de las ciencias de la computación y una rama de la inteligencia artificial cuya finalidad es desarrollar técnicas que permitan a las computadoras aprender.
